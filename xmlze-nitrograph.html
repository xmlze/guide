About

Ringkasan Proyek

Nitrograph adalah Layer-1 blockchain khusus untuk AI agents otonom, memungkinkan mereka discover, negotiate, dan transact cepat & murah. Analoginya: seperti jalan tol untuk robot AI yang bekerja mandiri di ekonomi digital.

Details

  • Funding: Rumor $5M dari Google (belum confirmed).
  • Backers: Google (rumor).
  • Status: Testnet Beta.
  • Tokenomics: 25% alokasi untuk community (airdrop).
  • Mainnet ETA: 2026.

Workflow — Langkah Kontribusi

1. Connect Wallet

Masuk ke situs community dan hubungkan wallet testnet.

  • Kunjungi https://community.nitrograph.com/
  • Hubungkan wallet (EVM-compatible).

2. Masuk Referral Code

Gunakan code untuk join, contoh: V7NOD37F.

  • Input code di dashboard.
  • Verifikasi.

3. Request Test Tokens

Ambil tokens dari faucet untuk test.

  • Kunjungi https://faucet-testnet.nitrograph.foundation/
  • Claim tokens.

4. Claim Daily $NITRO

Claim harian untuk poin airdrop.

  • Login daily.
  • Klik claim.

5. Complete Missions

Selesaikan tugas seperti deploy swarm.

  • Cek missions di dashboard.
  • Submit proof.

6. Invite Friends

Bagikan code untuk extra poin.

  • Generate referral.
  • Share ke teman.
