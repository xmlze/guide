Persiapan Singkat

  • Install wallet Solana‑compatible: Phantom / Solflare.
  • Tambahkan RPC testnet Fogo (lihat Docs) & connect wallet ke testnet.
  • Claim token testnet via faucet (Docs & channel #faucet di Discord).
  • Gabung Discord & follow X: penting untuk pengumuman Flames.

Step by step: Dari Nol sampai Aktif

  1. Claim Faucet — buka Docs & faucet; klaim test token ke wallet kamu. (Simpan tx hash.)
  2. Set RPC & Connect Wallet — tambahkan endpoint RPC Testnet di Phantom / Solflare sesuai Docs.
  3. Coba dApps Utama — lakukan transaksi nyata di: Valiant Swap (swap & LP), FluxBeam (perps/limit jika ada), Pyron (lending/borrow), dan RugCheck. Lakukan 2–3 action berbeda per hari.
  4. Catat & Backup — tiap interaksi simpan tx hash + waktu + screenshot. Ini bukti bila tim perlu verifikasi.
  5. Ikut Flames — pastikan men CLAIM / register di halaman Flames jika diperlukan; ikuti aturan Season aktif (baca rules di halaman Flames).
  6. Lapor Bug — kalau nemu error, pakai template di bawah dan kirim ke #bug-report atau GitHub issues (jika repos tersedia).
  7. Share & Bangun Reputasi — thread X, post Discord (buat summary mingguan), dan tutorial singkat supaya tim notice kontribusi kamu.

Template Laporan Bug (Paste ke Discord / GitHub)

Title: [BUG]  — ex: Swap gagal di Valiant
Date/Time (UTC+7): 2025-09-20 15:30
Wallet: Phantom (addr: 0x...)
dApp: Valiant Swap (Testnet)
Action: Swap 10 TESTFOGO -> fUSD (slippage 1%)
Expected: Swap success
Result: TX failed, error: "timeout" / "insufficient liquidity"
RPC endpoint: https://rpc.testnet.fogo.io
Blockhash / Tx: 
Screenshot / Logs: [attach]
Steps to reproduce:
  1. Connect wallet
  2. Select pair TESTFOGO-fUSD
  3. Input 10 TESTFOGO
  4. Confirm -> TX fails
Impact: High (blocks swaps for small users)
Suggested severity: Medium

Tips Anti‑Banned (Rules Flames)

  • Jangan create multiple accounts untuk farming.
  • Hindari bot/automated scripts yang menghasilkan pola inorganik.
  • Periksa rules Season (limit per minggu / per wallet).
  • Simpan bukti aktivitas: tx hash, screenshot, time stamp.

Prioritas Kontribusi Bernilai Tinggi

  1. Laporkan bug performa & validator zonal (latency, fork, fallback consensus).
  2. Stress test UX dApps (add/remove liquidity, partial fills, slippage edge cases).
  3. Buat tutorial publik: cara deploy kontrak/test scripts di Fogo (early adopter gets noticed).
